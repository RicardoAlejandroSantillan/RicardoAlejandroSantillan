# About Me

**Desarrollador de software orientado a análisis de datos y automatización**

Convierto datos desordenados en conocimiento útil. Mi base en **.NET**, **SQL** y **Python** me permite construir aplicaciones robustas y limpias, automatizar tareas y traducir métricas en decisiones. Soy un analista práctico y obsesivo del orden: limpio datos, modelo indicadores y automatizo procesos para entregar resultados claros y medibles. Hoy enfoco mi perfil en *Data Analytics* y automatizaciones, combinando ingeniería y ciencia de datos para maximizar el impacto.

## Public repositories

### Aplicaciones industriales y de automatización

- **DaikinEtiquetas** – Sistema de impresión y gestión de etiquetas para procesos industriales de Daikin. Permite crear y administrar etiquetas personalizadas para diferentes líneas de producción. Tecnologías: C#, Windows Forms, SQL Server.
- **MasterLabel** – Herramienta para crear, editar y gestionar etiquetas en proyectos de automatización. Facilita el control de versiones y la generación de plantillas consistentes. Tecnologías: .NET, WPF, SQL Server.

### Proyectos web y comercio electrónico

- **devPortfolio** – Portafolio web profesional para mostrar proyectos, habilidades y experiencia. Incluye navegación por vistas parciales, traducción ES/EN y carrusel de proyectos. Tecnologías: ASP.NET MVC, JavaScript, HTML, CSS, jQuery.
- **Pottery‑Shop** – Aplicación web de comercio electrónico enfocada en la venta de productos de cerámica. Incluye catálogo, carrito de compras y sistema de autenticación. Tecnologías: HTML, CSS, Bootstrap, JavaScript.
- **spaPage** – Single Page Application de ejemplo para consumo de APIs. Tecnologías: React, JavaScript, CSS.
